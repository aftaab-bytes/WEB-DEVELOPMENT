COLOUR PROPERTIES
The "color" is used to give a rule for text color

TYPES OF COLOR CLASSIFICATION


RGB (RED,GREEN,BLUE) eg:-rgb(40,50,14)
HEX code eg:-#ff7180
HSL (HUE,SATURATION,LIGHTNESS) eg:-hsl(8%,15%,60%)

These are the values used to specify the color for the text or the background based on the requirement
We also have "rgba" where a stands for alpha but these are used rarely.
