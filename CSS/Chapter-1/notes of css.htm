THIS ARE SOME TYPE OF SELECTIONS DONE IN CSS TO DO STYLING

The Element Selector

This is the selection of the particular element in the webpage when the element is targeted then all the contents in the element gets gets styled accordingly

The Class Selector

This is the selection of the particular class in the webpage when the class is targeted then all the contents in the class gets gets styled accordingly

Id Selector

This is the selection of the particular Id in the webpage when the Id is targeted then all the contents in the Id gets gets styled accordingly
Note :- id should be used only once in the webpage

Group Selector

We can do classing in groups so that the group of tag can get a same style

Ch-02 COLOR PROPERTIES
The "color" is used to give a rule for text color

TYPES OF COLOR CLASSIFICATION

RGB (RED,GREEN,BLUE) eg:-rgb(40,50,14)
HEX code eg:-#ff7180
HSL (HUE,SATURATION,LIGHTNESS) eg:-hsl(8%,15%,60%)

These are the values used to specify the color for the text or the background based on the requirement
We also have "rgba" where a stands for alpha but these are used rarely.

BACKGROUND COLOR

We use "background-color" to define the rules for background color
such as background-color: black; we get the background color as black.

BACKGROUND-IMAGE

We use "background-image" to set an image in the background 

For eg:-    body{
    background-image: url("GOLD-FISH.jpeg");
}

BACKGROUND-REPEAT

We use this property to repeat the image in x and y directions
